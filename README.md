# Med Agent – Análise de Imagens Médicas

Este projeto utiliza LLM Multimodal para analisar imagens médicas e retornar informações úteis ao profissional de saúde.

## Como funciona
- O usuário faz o upload de uma imagem
- O modelo processa e gera uma análise do conteúdo

## Tecnologias
- Python
- Gradio (Interface Web)
- Google Generative AI (modelo)
- Pillow (manipulação de imagens)

## Como executar localmente
```bash
pip install -r requirements.txt
python app.py
